ics Google Calender to Excel/CSV

1. Use browser that support ES6

Please use the latest version of Google Chrome atau Firefox

2. Export the calendar as a .zip file

Open Google Calendarclick Settings on the top right→Calendar Settings→Export Calendar.

3. Unzip the .zip to get the .ics .ics file

Any decompression tool will do, the open source free 7-zip is recommended.

Each .ics file corresponds to each of your calendars.

4. Select the .ics file from the button below


6. Can be downloaded as Excel file

5. The first 10 records will be displayed here

Source code on Github. If you have any questions or needs, please modify or report to Github.
